# تطبيق الخدمات المنزلية

تطبيق Flutter للخدمات المنزلية باللغة العربية.

## المميزات

- 🏠 الصفحة الرئيسية مع شريط البحث والفئات
- 👥 عرض أفضل مقدمي الخدمة
- 📍 إدارة العناوين المحفوظة
- 🔍 صفحة البحث والفلترة
- 📱 قائمة جانبية مع جميع الخيارات
- 🎨 تصميم عصري بألوان خضراء

## التشغيل

```bash
flutter pub get
flutter run
```

## البنية

```
lib/
├── main.dart                 # نقطة البداية
├── screens/
│   ├── home_screen.dart      # الصفحة الرئيسية
│   ├── addresses_screen.dart # صفحة العناوين
│   └── search_screen.dart    # صفحة البحث
└── widgets/
    ├── category_item.dart    # عنصر الفئة
    └── service_provider_card.dart # بطاقة مقدم الخدمة
```
